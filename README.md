# TepkiNet
TepkiNet, kullanıcıların şirketlere yönelik şikayetlerini oluşturabildiği, şirketlerin bu şikayetleri yönetip yanıtlayabildiği ve yöneticilerin platformu denetleyebildiği çok rollü bir şikayet yönetim sistemidir. Spring Boot, React, PostgreSQL ve JWT tabanlı kimlik doğrulama kullanılmıştır.
